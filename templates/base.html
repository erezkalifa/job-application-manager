How To Use

Getting Started

Welcome to Job Applications Manager! This tool helps you track your job applications and manage the interview process effectively.

Managing Applications

1. Add new job applications using the "Add Job" button
2. Track application status (Draft, Applied, Interviewing, etc.)
3. Upload and manage different versions of your resume
4. Track interview stages and questions

Interview Tracking

For each job, you can:
1. Add multiple interview stages
2. Record scheduled dates and outcomes
3. Save interview questions and your answers
4. Track key takeaways from each stage
